## 🧠 3-SAT Quick Recap (For Readers)

The **3-SAT problem** asks:  
Given a Boolean formula in conjunctive normal form (CNF), with each clause having exactly 3 literals, can we assign True/False to variables to make the entire formula true?

**Example (unsatisfiable)**:
\[
(A \lor B \lor C) \land (\lnot A \lor \lnot B \lor \lnot C) \land (A \lor \lnot B \lor C)
\]

Each clause is a disjunction (OR), the whole formula is a conjunction (AND).

---

## 🔁 How LLM Agents Typically Fail

When tasked with this in natural language or code, LLMs:
- Assign variables greedily without considering future contradictions.
- Lack a consistent memory of previous assignments.
- Fail to revise Ψ when Φ violates constraints.
- Output partially correct solutions that don’t satisfy all clauses.

In your φ⁰ loop:
- **Ψ**: The agent’s current memory of variable assignments and tested clauses.

## ✍️ Draft φ⁰ Function for 3-SAT

```python
def evaluate_clause(clause, assignments):
    for literal in clause:
        var = literal.replace('¬', '')
        is_negated = literal.startswith('¬')
        if var in assignments:
            val = not assignments[var] if is_negated else assignments[var]
            if val:
                return True
    return False

def phi0(Psi, Phi, clauses):
    # Merge candidate assignment into memory
    new_Psi = Psi.copy()
    new_Psi.update(Phi)
    
    # Validate new memory against clauses
    for clause in clauses:
        if not evaluate_clause(clause, new_Psi):
            print(f"Incoherent with clause {clause}. Backtracking.")
            return Psi  # Reject update
    
    print(f"Accepted update: {Phi}")
    return new_Psi
```
